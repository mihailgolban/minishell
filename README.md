# Minishell

My implementation of [minishell](minishell.en.pdf). As beautiful as a shell

### Goals
Create the simplest start of a shell script.

## Installation

### Requirements
* GNU make
* GNU C Compiler (GCC)

## Usage
```
git clone https://github.com/mihailgolban/minishell.git
cd minishell
make
./minishell
```
